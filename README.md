
Install
============================

```
git clone --depth 1 https://github.com/kanalun/markdown2html.git
cd markdown2html
npm -g install
markdown2html aaaa.md > aaaa.html
```
Usage
============================
```
markdown2html aaaa.md > aaaa.html
```
OR
```
md2html aaaa.md > aaaa.html
```
OR
```
node index.js aaaa.md > aaaa.html
```
